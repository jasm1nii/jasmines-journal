static sites only

mmm.page
  • minimal coding knowledge required with a drag and drop site editor
  • does not support exporting site files nor social discovery (yet)
hotglue
  • also a low-code site editor
  • sites are discoverable via feed
  • provides community forums for troubleshooting and other inquiries
neocities
  • free tier - 1 GB storage
  • paid tier - 50 GB storage, custom domains (purchased separately), no file restrictions
  • having a social profile is optional, but sites will always be discoverable via search engines or other means
  • supports site management via command line and API
ichi.city
  • no email requried for account creation (which means no way of recovering a lost password)
  • no paid tier
  • 10 MB storage limit
  • all sites are discoverable via global feed
  • no social following features
  • supports site management via SFTP (secure file transfer protocol)
vistaserv.net

static and dynamic sites

glitch

supports a wide range of sites, from static link pages to full-stack apps with node.js, react, and SQlite

teacake

supports PHP, mySQL, apache, and FTP - paid plans also come with cPanel and softalicious scripts

leprd

completely free hosting with higher storage limits than teacake, but do note that slots are much more limited
