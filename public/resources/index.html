notion üëë üéÄ

this is my go-to app for... almost everything. i basically use it as a second brain.

it's a note-taking app that can be as simple or as powerful as you need. some use cases from my everyday life:

  • vent journalling
  • cataloguing web bookmarks in a browser-agnostic system - my favorite way to use notion
  • keeping track of school assignments and making detailed project timelines
  • building private wikis with friends
  • remembering things in general

also, you can export the pages as HTML files! i've yet to use them for web weaving myself, but one could potentially use notion notes as web templates. and with markdown support, drafting blogs would be much easier, too.

i've only used it sparingly, but here are the things that stick out, versus notion:

pros

  • much cheaper to scale for teams, with content limits per page (instead of per team)
  • native form-collection feature
  • easier to onboard others for group projects (in my experience)

cons

  • the mobile app leaves much to be desired
  • feels a bit overengineered for individual use
